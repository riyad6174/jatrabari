# **App Name**: Veritas College

## Core Features:

- Homepage Navbar: Navigation bar with links to key sections (Home, About Us, Academics, Admission, Gallery).
- Homepage Banner: A visually appealing banner image to capture attention.
- Notice Slider: Scrolling section for displaying important notices.
- Academic Sections: Sections showcasing information about the college, chairman's message, principal's message, vision statement, etc.
- Teacher Slider: Interactive slider displaying teacher cards with name, image, designation, and 'View Profile' button.
- Gallery Slider: Dynamic gallery slider showcasing college images and activities.
- Teacher Profile Modal: A modal that displays detailed information about a teacher when 'View Profile' is clicked.

## Style Guidelines:

- Primary color: Dark blue (#003366) to convey professionalism and trust.
- Background color: Light gray (#F0F0F0), offering a clean backdrop that complements the dark blue primary.
- Accent color: A slightly lighter shade of blue (#4682B4), to guide the user and highlight areas of importance. 
- Body text: 'PT Sans' sans-serif for clarity.
- Headline text: 'Space Grotesk' sans-serif for impactful headings; where longer text is used below the headline, PT Sans will be used for body text
- Use professional and clear icons relevant to the different sections and information presented.
- Grid-based layout for a structured and responsive design.
- Cards with sharp, non-rounded corners and subtle box shadows for a modern aesthetic.
- Smooth transitions and subtle animations for a polished user experience.
